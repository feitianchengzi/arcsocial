# Content Workflow

## 1. Capture

Put raw thoughts, references, screenshots, and materials into `inbox/`.

## 2. Select

Promote useful inputs into `content/drafts/` when they have a clear audience or angle.

## 3. Develop

Use the draft template to clarify the core idea, supporting material, and canonical copy.

## 4. Adapt

Create platform-specific versions under `platforms/` or `content/ready/`.

## 5. Publish

Use `playbooks/checklists/publish-checklist.md` before posting.

## 6. Learn

Record metrics, comments, and follow-up ideas in `data/metrics/` and `playbooks/experiments/`.
